# Doctor Consult WordPress Theme

A modern, responsive WordPress theme designed for healthcare and medical consultation websites, featuring a PharmEasy-inspired design.

## Features

- **Responsive Design**: Mobile-first approach with tablet and desktop optimizations
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Healthcare Focused**: Designed specifically for medical consultation services
- **WordPress Standards**: Follows WordPress coding standards and best practices
- **Customizable**: Easy to customize colors, fonts, and layouts

## Theme Structure

```
doctor_consult/
├── style.css          # Main stylesheet with theme information
├── index.php          # Main template file
├── header.php         # Header template
├── footer.php         # Footer template
├── functions.php      # Theme functions and setup
├── js/
│   └── script.js      # Theme JavaScript
└── README.md          # This file
```

## Installation

1. Upload the theme folder to `/wp-content/themes/`
2. Activate the theme through the WordPress admin panel
3. Customize the theme through the WordPress Customizer

## Customization

The theme supports the following customization options:

- Logo upload through WordPress Customizer
- Navigation menu registration
- Custom color schemes (easily modifiable in style.css)
- Responsive breakpoints

## Browser Support
